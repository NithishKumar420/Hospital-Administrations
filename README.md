# Hospital Administration System

A web-based Hospital Administration System built using Flask.  
This project helps manage hospital staff, appointments, users, and website settings.

## 🚀 Features
- Admin Login
- Staff Management
- Appointment Booking
- User Management
- Website Settings
- Admin Dashboard

## 🛠 Tech Stack
- Python
- Flask
- MySQL
- HTML
- CSS

## 📂 Project Structure
Hospital-Administrations
│
├── app.py
├── models.py
├── migrations/
├── static/
├── templates/
└── database
